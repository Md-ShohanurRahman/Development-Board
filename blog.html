Question-1: What are the different ways to select an element in the DOM?

----------------------------------------------------------------------------------------------------------------------------------------

There are several ways to select an element in the DOM (Document Object Model). The most common methods include:

  1. 1.getElementById(): Selects an element by its unique ID.
  2. 2.getElementById(): Selects an element by its unique ID.
  3. 3.getElementsByClassName(): Selects elements with a specific class name.
  4. 4.getElementsByTagName(): Selects elements with a specific tag name.
  5. 5.querySelector(): Selects the first element that matches a CSS selector.
  6. These methods allow you to target elements based on their ID, class, tag, or CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

----------------------------------------------------------------------------------------------------------------------------------------

The difference between innerHTML, innerText, and textContent lies in how they handle the content of an element:

  1. 1.innerHTML: Returns or sets the HTML content inside an element, including any HTML tags. It can also modify the HTML structure, such as adding or removing elements.
  2. 2.innerText: Returns or sets the visible text inside an element, excluding any HTML tags. It takes CSS styling into account, such as hidden text or text hidden by display: none.
  3. 3.textContent: Returns or sets the text content inside an element, but unlike innerText, it does not take CSS styles into account. It simply gives or changes the raw text, including hidden text.
  4. In summary, innerHTML handles HTML, innerText focuses on visible text, and textContent deals with all text, visible or not.
